🤝 即時協作

需要先設定 Firebase
協作功能需要 Firebase Realtime Database。
請參考下方步驟完成設定。
  1. 前往 console.firebase.google.com 建立免費專案
  2. 新增 Web App（</> 圖示），複製 firebaseConfig
  3. 進入 Realtime Database → 建立資料庫（選測試模式）
  4. 將設定值填入專案中的 firebase-config.js
  5. 重新整理頁面即可使用協作功能
建立協作房間

將目前行程上傳並產生一組房間碼，傳給朋友即可一起編輯。
